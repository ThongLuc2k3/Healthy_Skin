## File da tao

- `CHECKLIST_THUC_THI_CTP2026.md`: checklist ngan gon theo thu tu uu tien

- `WS5_KIEM_THU_TU_DONG.md`: ghi chu kiem thu tu dong va blocker moi truong
- `WS7_OUTLINE_PITCH_DECK.md`: outline pitch deck toi da 11 slide
- `WS8_KICH_BAN_VIDEO_90S.md`: kich ban video 90 giay

## Gioi han hien tai

- Chua co web access trong phien nay, nen cac task `WS1` va phan gia dich vu hien hanh cua `WS6.2`
  chua duoc dien bang so lieu moi.
- `node` va `npm` khong co trong PATH cua moi truong shell hien tai, nen chua the chay `test`, `lint`,
  `build` tu day. Da ghi lai trong `WS5_KIEM_THU_TU_DONG.md`.

## Thu tu de tiep tuc

1. Dien thong tin that con thieu: ma nhom, ten thanh vien, gia dinh kinh doanh, quyet dinh nguon thu.
2. Khi co internet, lam `WS1` va cap nhat `WS6`.
3. Khi co `node`/`npm`, chay lai `npm run test`, `npm run lint`, `npm run build`.
